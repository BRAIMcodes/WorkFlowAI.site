1. **Download Blueprint:** Click the download button above to get the raw JSON file from the GitHub repository.

3. **Configure Node Credentials:** Double-click each node representing Local File Trigger, LangChain, Set, Merge, Split Out, Split In Batches, Aggregate, Wait, Convert To File, Read Write File, Switch, Extract From File, Sticky Note and authenticate with your account credentials.
4. **Activate Scenario:** Test the flow manually by clicking **Test step** on the trigger, verify the output, then toggle the workflow to **Active** to start running.
