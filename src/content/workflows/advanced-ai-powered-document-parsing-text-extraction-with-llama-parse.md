3. **Configure Node Credentials:** Double-click each node representing Webhook, Gmail, Limit, Sticky Note, Http Request, Google Sheets, Google Drive, Set, Telegram, If, Aggregate, Merge and authenticate with your account credentials.
4. **Activate Scenario:** Test the flow manually by clicking **Test step** on the trigger, verify the output, then toggle the workflow to **Active** to start running.
