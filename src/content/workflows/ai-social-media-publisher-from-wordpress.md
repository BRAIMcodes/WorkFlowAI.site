3. **Configure Node Credentials:** Double-click each node representing Manual Trigger, Google Sheets, Sticky Note, Wordpress, Twitter, Linked In, Facebook Graph Api, Http Request and authenticate with your account credentials.
4. **Activate Scenario:** Test the flow manually by clicking **Test step** on the trigger, verify the output, then toggle the workflow to **Active** to start running.
